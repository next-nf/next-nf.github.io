# next-nf.github.io

Source for <https://next-nf.github.io/> — the landing page for the **next-nf**
project: open-source 3GPP mobile-core network functions in Erlang/OTP.

Built with GitHub Pages' native Jekyll (`jekyll-seo-tag` + `jekyll-sitemap`).
Edit `index.md` for content and `_config.yml` for site metadata; GitHub rebuilds
on push to `main`.
